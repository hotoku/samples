## ライブラリのインストール

```
$ npm i -D webpack webpack-cli typescript ts-loader
$ npm i -S react react-dom @types/react @types/react-dom
```

## 必要なファイル

- package.json
- tsconfig.json
- webpack.config.js

## ビルドコマンド

上のファイルを適当なディレクトリにコピーして、以下を実行する。

```
$ npm run build
```
